基础知识

变量

变量用来存储具体的值，使用关键字let或var来定义变量：

let a;
var b;

赋值

使用等号表示将右侧的值赋给左侧的变量。

a = "老白";
b = "小白";

从 0 开始

自古以来，人们都习惯用 1 来代表计数的开始。

从 20 世纪 60 年代中期开始，一小股有影响力的程序员认为，计数应当从 0 开始。到了今天，几乎所有程序员都习惯了这种以 0 开始的计数法。

目前，主流编程语言都是从 0 开始计数，虽然很奇怪，习惯了也就好了。

let staff = ["佟湘玉", "白展堂", "莫小贝"];

console.log(staff[0]);
