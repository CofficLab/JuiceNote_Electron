输出结构体

Go语言格式键值

官方文档原文是：a Go-syntax representation of the value。

也就是说，输出的是一串 Go 语言代码。

package main
import "fmt"

type Dog struct {
    name string 
    age int
}

func main() {
    dog := Dog{"cici",1}
    
    fmt.Printf("%#v", dog)
}

Go语言格式类型

官方文档原文是：a Go-syntax representation of the type of the value

package main
import "fmt"

type Dog struct {
    name string 
    age int
}

func main() {
    dog := Dog{"cici",1}
    fmt.Printf("%T", dog)
}

仅输出值

package main
import "fmt"

type Dog struct {
    name string 
    age int
}

func main() {
    dog := Dog{"cici",3}

}

输出错误

fmt.errorf 等价于 errors.New(fmt.Sprintf())

前补齐

package main
import "fmt"

func main() {
    fmt.Printf("->%20s<-", "***")
}

后补齐

package main
import "fmt"

func main() {
    fmt.Printf("->%-20s<-", "***")
}
