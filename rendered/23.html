fmt 是 Go 标准库中提供的包，Go 主要使用 fmt 包来进行输出，功能非常强大。

换行输出

package main
import "fmt"

func main() {
    // Println 是 Print Line 的简写，意思是：输出一行
    fmt.Println("老白，吃了吗？")
    fmt.Println("老邢，吃了吗？")
}

不换行输出

package main
import "fmt"

func main() {
    fmt.Print("老白，吃了吗？")
    fmt.Print("老邢，吃了吗？")
}

格式化输出 Printf

package main
import "fmt"

func main() {
    var name string = "小六"

    // Printf 是 Print Format 的简写，意思是：格式化输出
    fmt.Printf("吃了吗？ %s", name)
}

除了 %s，还有更多表示其他意义的占位符，在此列举部分：

类型 含义 %b 将整数表示为二进制数 %d 将整数表示为十进制数 %e 使用科学计数法来表示小数 %% 一个百分号 %T %v ... ...

格式化返回 SPrintf

如果仅需要返回值，不需要直接输出，则使用 SPrintf。

package main
import "fmt"

func main() {
    var name string = "小六"

    // Sprintf-&gt;String Print Format-&gt;返回格式化后的字符串
    var string = fmt.Sprintf("吃了吗？ %s", name)

    fmt.Println(string)
}

换行输出

package main
import "fmt"

func main() {
    fmt.Println("老白，\n吃了吗？")
}

制表符输出

package main
import "fmt"

func main() {
    fmt.Println("小六吃了吗？")
    fmt.Println("小六\t吃了吗？")
}

这里的 t 是 table 的简写，制表符主要用于对齐表格的列：

package main
import "fmt"

func main() {
    fmt.Println("学号    \t姓名 \t语文 \t\t数学 \t\t英语")
    fmt.Println("2017001\t兰妃 \t99  \t\t88   \t\t0")
    fmt.Println("2017002\t花花 \t92  \t\t45   \t\t93")
    fmt.Println("2017008\t小溪 \t77  \t\t82   \t\t100")
}
