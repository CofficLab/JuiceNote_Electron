迭代

使用 for

要遍历字典的所有关键字，可像遍历序列那样使用普通的 for 语句。

<div class="run"></div>

d = {'x': 1, 'y': 2, 'z': 3}

for key in d:
    print(key, '=', d[key])

或

<div class="run"></div>

d = {'x': 1, 'y': 2, 'z': 3}

for key, value in d.items():
    print(key, '=', value)

并行迭代

<div class="run"></div>

names = ['佟湘玉','白展堂','莫小贝']
ages = [25,24,9]

for i in range(len(names)):
    print(names[i], '的年龄是', ages[i])

也可以用 zip 函数将两个列表“缝合起来”:

<div class="run"></div>

names = ['佟湘玉','白展堂','莫小贝']
ages = [25,24,9]
stuff = zip(names, ages)

for name,age in stuff:
    print(name, '的年龄是', age)
