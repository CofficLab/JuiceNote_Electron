复数类型

复数类型在实践中比较少见。

Go 内置的复数类型有两种：

  • complex64

  • complex128

复数在计算机里使用两个浮点数表示，一个表示实部，一个表示虚部。

complex64 是由两个 float32 构成的；
complex128 是由两个 float64 构成的。

复数的字面量表示和数学表示法一样。

var a complex64 = 3.1 + 5i
b := 3.1 + 6i

Go 有三个内置函数处理复数：

# 构造一个复数
var v = complex(2.1, 3)
# 返回复数实部
a := real(v)
# 返回复数虚部
b := image(v)
