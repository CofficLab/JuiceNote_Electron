# %

<div class="o">https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Operators/Remainder</div>

取余（%）运算符返回左侧操作数除以右侧操作数的余数。它总是与被除数的符号保持一致。

<div class="run"></div>

```JavaScript
console.log(13 % 5);
// Expected output: 3

console.log(-13 % 5);
// Expected output: -3

console.log(4 % 2);
// Expected output: 0

console.log(-4 % 2);
// Expected output: -0
```
