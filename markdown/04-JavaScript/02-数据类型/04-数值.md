# 数值

在 JavaScript 中，数值的存储方式为 64 位的浮点小数。

所以，所有的数值都是浮点小数。

## 判断类型

<div class="run"></div>

```JavaScript
var a = 1;
var b = 1.0;

console.log(typeof a);
console.log(typeof b);
```

为了理解 JavaScript 的不同，你可以看看其他语言对整数和小数是怎么处理的。

## 运算

<div class="run"></div>

```JavaScript
var a = 1;
var b = 2;

console.log(a + b);
```

## 复数类型

JavaScript 没有内置复数类型。
