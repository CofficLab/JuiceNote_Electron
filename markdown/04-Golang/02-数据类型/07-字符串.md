# 字符串

Go 将字符串作为一种原生的基本数据类型，字符串的初始化可以使用字符串字面量。

例如：

```go
var a = "hello"
```

## 索引

字符串是常量，可以通过类似数组的索引访问其子节单元，但是不能修改某个字节的值。

例如：

<div class="run"></div>

```go
package main
import "fmt"

func main() {
    var a = "hello"
    fmt.Print(a[0])
    a[1] = 'a'
}
```

## 拼接

<div class="run"></div>

```go
package main
import "fmt"

func main() {
    a := "hello"
    b := "老白"
    c := a + b
    fmt.Print(c)
}
```
