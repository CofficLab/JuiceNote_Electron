# 整型

整型、整数型、整数的意思相同或接近。

Go 内置了 12 种整数类型，这显得有些复杂。

你可以看看 Python 对整型是怎么处理的。

## 范围

整数：

- int8（-128 -> 127）
- int16（-32768 -> 32767）
- int32（-2,147,483,648 -> 2,147,483,647）
- int64（-9,223,372,036,854,775,808 -> 9,223,372,036,854,775,807）

无符号整数：

- uint8（0 -> 255）
- uint16（0 -> 65,535）
- uint32（0 -> 4,294,967,295）
- uint64（0 -> 18,446,744,073,709,551,615）

## 四则运算

<div class="run"></div>

```go
package main
import "fmt"

func main() {
	var a int = 1
	var b int = 2
	var c = a + b

	fmt.Print(c)
}
```
