## 什么是条件

条件是一种表达式，它的结果只有两个：

- true
- false

## 定义条件

### 值的比较

| 操作符 | 含义                           |
| ------ | ------------------------------ |
| ==     | 判断两个变量的值是否相等       |
| ===    | 判断两个变量类型和值是否都相等 |
| >      | 判断左侧是否比右侧大           |
| >=     | 左是否大于等于右               |
| <      | 左是否小于右                   |
| <=     | 左是否小于等于右               |
| !=,<>  | 左是否不等于右                 |
| !==    | 左右类型和值是否都不相等       |
