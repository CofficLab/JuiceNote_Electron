函数 open 用来打开文件：

```python
f = open('names.txt')
```

如果文件位于其他地方，可指定完整的路径。

如果指定的文件不存在，将抛出异常:

<div class="run"></div>

```python
f = open('/a_file_that_not_exists.txt')
```

为了保证程序的健壮性，我们应捕获异常。
