# 简介

Java 中有两种有效地数据类型：

- 原始数据类型
- 引用数据类型

## 原始数据类型

Java 支持 8 种原始数据类型。

| 数据类型     | 英文    | 位数 |
| ------------ | ------- | ---- |
| 字符类型     | byte    | 8    |
| 短整型       | short   | 16   |
| 整型         | int     | 32   |
| 长整型       | long    | 64   |
| 单精度浮点型 | float   | 32   |
| 双精度浮点型 | double  | 32   |
| 字符型       | char    | 64   |
| 布尔型       | boolean | 1    |

## 引用数据类型

引用数据类型是由类的编辑器定义的。他们是用于访问对象的。这些变量被定义为不可更改的特定类型。例如：Employee， Puppy 等等。  
类对象和数组变量就是这种引用数据类型。  
任何引用数据类型的默认值都为空。  
一个引用数据类型可以被用于任何声明类型和兼容类型的对象。  
例如：

```java
Animal animal = new Animal("giraffe")；
```
