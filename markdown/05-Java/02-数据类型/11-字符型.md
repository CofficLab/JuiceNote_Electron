# 字符型

<div class="run"></div>

```java
public class Test {
    public static void main(String args[]) {
        char a = '中';
        char b = '国';
        char c = '人';

        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
    }
}
```

`中国人` 不是一个字符型，是三个字符型组成的 `字符串型`，这样会报错：

<div class="run"></div>

```java
public class Test {
    public static void main(String args[]) {
        char a = '中国人';

        System.out.println(a);
    }
}
```

字符的个数是有限的，字符串的个数是无限的。
