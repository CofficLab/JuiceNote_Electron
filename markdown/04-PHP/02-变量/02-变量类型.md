# 变量类型

## 数值

数值类型分为两种，整型和浮点型。

| 整形 | 浮点型 |
| ---- | ------ |
| 1    | 1.0    |
| 1972 | 19.72  |
| -1   | -1.0   |

## 字符串

一个字符串是一对单引号或者双引号括起来的任意个数的字符。

- "Hello"
- "Hello, $name"
- "1/3"
- 'Nice to meet you'
- "1996"
- ''

## 数组

数组是值的列表。

可以用正整数作为键：
| 键 | 值 |
| --- | --- |
| 0 | a |
| 1 | b |
| 2 | c |

也可以用字符串作为键：

| 键   | 值      |
| ---- | ------- |
| name | Li Ming |
| age  | 22      |
