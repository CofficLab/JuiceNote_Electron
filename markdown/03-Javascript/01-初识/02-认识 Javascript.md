# 认识 Javascript

Python 是什么？有什么特点？

- 一种可以用来给网页增加交互的高级编程语言；
- 也可以用来做网页的服务端。

## 特点

- 代码简洁，清晰易懂
- 应用广泛

## Javascript 与 Java

Javascript 与 Java 没有关系。
