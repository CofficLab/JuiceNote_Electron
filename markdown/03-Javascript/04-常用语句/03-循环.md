# 循环

## while 循环

<div class="run"></div>

```javascript
let x = 1;
while (x <= 10) {
  console.log(x);
  x = x + 1;
}
```

## for 循环

<div class="run"></div>

```javascript
for (i = 1; i <= 10; i = i + 1) {
  console.log(i);
}
```

### do...while 循环

<div class="run"></div>

```javascript
let i = 1;
do {
  console.log(i);
  i = i + 1;
} while (i <= 10);
```
