# 条件判断

## if...else

<div class="run"></div>

```javascript
let password = "123";

if (password == "123") {
  console.log("密码正确");
} else {
  console.log("密码错误");
}
```

## switch...case

<div class="run"></div>

```javascript
let type = "book";

switch (type) {
  case "book":
    console.log("图书");
    break;
  case "chapter":
    console.log("章节");
    break;
  case "page":
    console.log("页面");
    break;
  default:
    console.log("未知");
}
```
