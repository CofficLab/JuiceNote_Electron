# 数组

数组是最常用的数据结构。

## 定义数组

<div class="run"></div>

```javascript
let stuff = ["佟湘玉", "白展堂", "莫小贝"];

console.log(stuff);
```

## 长度

每个数组都有一个 `length` 属性。

<div class="run"></div>

```javascript
let stuff = ["佟湘玉", "白展堂", "莫小贝"];

console.log(stuff.length);
```

## 删除

<div class="run"></div>

```javascript
let stuff = ["佟湘玉", "白展堂", "莫小贝", "刑育森"];

delete stuff[3];

console.log(stuff);
```

## 枚举

<div class="run"></div>

```javascript
let stuff = ["佟湘玉", "白展堂", "莫小贝"];

for (let i = 0; i < stuff.length; i = i + 1) {
  console.log(stuff[i]);
}
```
