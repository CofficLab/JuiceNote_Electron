# 数据类型

## 常见的数据类型

- 整型
- 浮点型
- 字符串型
- 布尔型
- 时间日期型

## 为变量指定数据类型

PHP 是一种弱类型的脚本，你不需要刻意的为变量指定数据类型。

PHP 处理引擎可以根据变量的值自动判断该变量的数据类型，同时还会根据实际情况来变更数据的类型。
