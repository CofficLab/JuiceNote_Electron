# 魔术方法

## 构造函数

<div class="run"></div>

```python
class FooBar:
    def __init__(self):
        self.name = '同福'
f = FooBar()
print(f.name)
```

函数 `__init__` 称为构造函数，对象创建后会自动调用它。

给构造函数添加参数:

<div class="run"></div>

```python
class FooBar:
    def __init__(self, name='同福'):
        self.name = name

k = FooBar()
f = FooBar('龙凤')

print(k.name)
print(f.name)
```

## 其他魔术方法

| 方法        | 功能                     |
| ----------- | ------------------------ |
| **del**     | 析构函数，释放对象时使用 |
| **repr**    | 打印，转换               |
| **setitem** | 按照索引赋值             |
| **getitem** | 按照索引获取值           |
| **len**     | 获得长度                 |
| **cmp**     | 比较运算                 |
| **call**    | 函数调用                 |
| **add**     | 加运算                   |
| **sub**     | 减运算                   |
| **mul**     | 乘运算                   |
| **div**     | 除运算                   |
| **mod**     | 求余运算                 |
| **pow**     | 乘方                     |
