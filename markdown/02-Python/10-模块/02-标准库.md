# 标准库

## sys

模块 sys 让你能够访问与 Python 解释器紧密相关的变量和函数。

<div class="run"></div>

```python
from sys import argv

print(argv)
```

<div class="run"></div>

```python
from sys import path

print(path)
```

<div class="run"></div>

```python
from sys import platform

print(platform)
```

## os

访问操作系统提供的服务。

<div class="run"></div>

```python
from os import environ

print(environ)
```
