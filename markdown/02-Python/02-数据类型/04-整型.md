# 整型

整型、整数型、整数的意思相同或接近。

Python 对整数的处理非常简单和易于理解：

## 四则运算

<div class="run"></div>

```python
print(2+2)
print(2-2)
print(2*3)
print(3/2)
```

为了进一步体会 Python 的简洁，你可以看看 Java 对这些是怎么处理的。

## 乘方运算

<div class="run"></div>

```python
print(3**2)
print(2**3)
print(2*3)
print(10**5)
```

## 运算次序

可以使用括号来表示运算次序，和平时数学中的用法一致：

<div class="run"></div>

```python
print((2 + 3) * 4)
```
