# 构造函数

<div class="run"></div>

```python3
class FooBar:
    def __init__(self):
        self.name = '同福'
f = FooBar()
print(f.name)
```

函数 `__init__` 称为构造函数，对象创建后会自动调用它。

给构造函数添加参数:

<div class="run"></div>

```python3
class FooBar:
    def __init__(self, name='同福'):
        self.name = name

k = FooBar()
f = FooBar('龙凤')

print(k.name)
print(f.name)
```
