# 简介

## 什么是数据

有意义的数字、一段文字，都可以认为是数据。

## 什么是数据结构

一些数据以某种关系聚集起来，就是：数据结构。

## 有哪些数据结构

数据结构并没有固定的数量，每个人都可以根据需要创造数据结构。

经过几十年的发展，形成了常用的几种数据结构：

- 链表
- 队列
- 栈
- 图
- 列表

不同的编程语言对这些数据结构有不同的实现和处理方式。

如果某一天，你发现这 4 种数据结构不能满足需求，完全可以自己创造一种数据结构。
