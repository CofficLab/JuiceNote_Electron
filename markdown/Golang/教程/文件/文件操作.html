常见的操作有以上几种。

对于这些非常常见的操作，不管是什么编程语言，都会内置。

对于 Go，你可以去标准库里寻找相应的函数。

标准库的官方地址：<a href="https://pkg.go.dev/std" target="_blank">https://pkg.go.dev/std</a>

创建

创建文件夹：

<div class="run"></div>

package main
import "fmt"
import "os"

func main() {
  // 先获取当前工作路径
  pwd, err := os.Getwd()

  if err != nil {
    fmt.Println("获取当前路径出错")
  } else {
    fmt.Println("当前工作路径：",pwd)

    target := pwd + "/temp/xxx.txt"
    err := os.WriteFile(target, []byte("Hello, 老白!"), 0666)
    if err != nil {
      fmt.Println("新建文件失败了，错误是：", err)
    } else {
      fmt.Println("新建文件成功了，路径是：", target)
    }
  }
}

查看文件列表

<div class="run"></div>

package main
import "os"
import "fmt"

func main() {
  // 先获取当前工作路径
  pwd, err := os.Getwd()

  if err != nil {
    fmt.Println("获取当前路径出错")
  } else {
    fmt.Println("当前工作路径：",pwd)

    files,err := os.ReadDir(pwd)

    if err != nil {
      fmt.Println("读取文件夹失败")
    } else {
      fmt.Print(files)
    }
  }
}

删除

删除一个文件

os.Remove(path)

删除空文件夹

os.Remove(path)

删除文件夹

删除文件夹以及其内容，不管是否为空：

os.RemoveAll(path)

文件夹会被彻底删除，不会到回收站，这是一个非常危险的操作。

读取

读取的前提是文件已经打开，可以指定读取多少个字符，也可以全部读取:

<div class="run"></div>

os.ReadFile(path)
