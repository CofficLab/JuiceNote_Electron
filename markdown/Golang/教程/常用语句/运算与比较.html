四则运算

大多数编程语言的工作方式都一样： - `+` 表示加法 - `-` 表示减法 - `*` 表示乘法 - `/` 表示除法

package main 
import "fmt" 
func main() { 
    fmt.Println(1 + 2) 
    fmt.Println(5.5 - 1.1) 
    fmt.Println(5 * 8) 
    fmt.Println(7.5 / 5) 
}

比较

大多数编程语言的工作方式都一样： - `>` 表示大于 - `<` 表示小于 - `>=` 表示大于等于 - `<=` 表示小于等于 - `==` 表示等于 比较的结果是一个布尔值。

```go package main import "fmt" func main() { fmt.Println(1 < 2) fmt.Println(1 > 2) fmt.Println(1 >= 2) fmt.Println(1 + 9 <= 2) fmt.Println(1 + 2 == 3) fmt.Println(1 + 2 != 3) } ```
