认识 Golang

Golang（简称 Go） 是一门非常年轻的编程语言。

啊

第一个 Go 程序

package main
import "fmt"

func main() {
    fmt.Println("hello")
}

每个编程语言的设计理念都不一样。

我们通过分析这段程序，来理解 Go 的设计理念。

包

https://pkg.go.dev/std

package main

每个 Go 源码文件的开头都必须有package声明，表示其中的代码所属的包。

Go 规定，我们写的程序必须有一个main包。

内置包

fmt

os

path

sort

time

zip

md5

png

...

内置包是 Go 本身就有的包，在程序的任何地方都能使用，比如：

fmt：处理输入输出

zip：压缩文件

png：处理png图片

内置包就像一个工具箱，里面放置着常用的工具。

这个内置包的集合的官方名称是：Standard Library，中文译为：标准库。

标准库里包含了哪些包？

这个答案在其官方文档里：https://pkg.go.dev/std

一定要学会查询官方文档

导入包

import "fmt"

如果你要使用一个包，就必须在程序的开头导入要使用的包。

fmt 是内置的非常常用的包，主要用于输出内容。

main 函数

func main() {
    fmt.Println("hello")
}

程序的main包里必须有一个main函数作为程序执行的起点。

这个main函数做的事是：

使用fmt包的Println函数，输出hello这串字。
