布尔

<div class="o">https://go.dev/ref/spec#Boolean_types</div>

布尔型（bool）只有两个值：

  • true

  • false

<div class="run"></div>

package main
import "fmt"
import "reflect"

func main() {
    var a bool = true

    fmt.Println(reflect.TypeOf(a), a)
    fmt.Println(10 == 5)
    fmt.Println(10 == 10)
}

布尔值常常用于条件判断中，比如：

<div class="run"></div>

package main
import "fmt"

func main() {
    // 括号内返回的是布尔值的 true
    if (1 + 3 == 4) {
        fmt.Println("回答正确")
    }
}
