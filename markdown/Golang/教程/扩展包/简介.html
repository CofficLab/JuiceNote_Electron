简介

在 Go 中，常用的程序逻辑有两种提供形式：

  • 内置函数

  • 内置包

标准库

包含了常用功能的包的集合称为标准库，官方网址是：<div class="link">https://pkg.go.dev/std</div>

导入包

使用import关键字导入包。

<div class="run"></div>

package main

import "fmt"

func main() {
  fmt.Println("老白，早上好")
}

导入多个时，可以这样：

import "fmt"
import "reflect"

也可以这样：

import (
  "fmt"
  "reflect"
)
