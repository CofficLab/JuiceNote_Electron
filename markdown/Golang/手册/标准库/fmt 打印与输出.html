fmt 打印与输出

o:https://pkg.go.dev/fmt@go1.20

通用

格式

含义

%v

默认格式

%+v

针对结构体，在 %v 的基础上输出结构体的键名

%#v

Go 语言语法格式的值

%T

Go 语言语法格式的类型和值

%%

输出 %, 相当于转义

整型

格式

含义

%b

二进制格式

%c

对应的 Unicode 码

%d

十进制

%o

八进制

%O

八进制，加上 0o 前缀

%q

Go 语言语法转义后的单引号字符 (很少使用) 例如 97 会输出 'a'

%x

十六进制 (小写), 例如 0xaf

%X

十六进制 (大写), 例如 0xAF

%U

Unicode 例如 "U+%04X"

Bool

格式

含义

%t

true 或 false

浮点型

格式

含义

%b

指数为 2 的幂的无小数科学计数法，例如 -123456p-78

%e

科学计数法，例如 -1.234456e+78

%E

科学计数法，例如 -1.234456E+78

%f

常规小数点表示法 (一般使用这个), 例如 123.456

%F

和 %f 功能一样

字符串

格式

含义

%s

字符串

%q

将双引号 " 转义后的字符串

%x

将字符串作为小写的十六进制

%X

将字符串作为大写的十六进制

指针

格式

含义

%p

地址的十六进制，前缀为 0x

场景

前补齐

package main
import "fmt"

func main() {
    fmt.Printf("->%20s<-", "***")
}

后补齐

package main
import "fmt"

func main() {
    fmt.Printf("->%-20s<-", "***")
}
