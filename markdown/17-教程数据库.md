## 渲染

这里的 Markdown 使用了 TailwindCSS 的 Class，以及 AlpineJS 的语法。  
比如可能会有这样的片段：

```
<div x-data="{ open: false }" class="bg-red-300">
    <button @click="open = true">Open Dropdown</button>
    <ul
        x-show="open"
        @click.away="open = false"
    >
        Dropdown Body
    </ul>
</div>
```

所以在渲染时，要保证加载了 AlpineJS 和 TailwindCSS。
