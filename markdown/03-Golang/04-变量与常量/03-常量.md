# 常量

常量就是在程序运行期间不会变的量。

如果强制改变它的值，将会报错。

<div class="run"></div>

```go
package main
import "fmt"
import "reflect"

func main() {
    // 可以省略类型，让 Go 自动推断
    const PI = 3.14
    const TEN int = 10

    fmt.Println(PI, reflect.TypeOf(PI))
    fmt.Println(TEN, reflect.TypeOf(TEN))
}
```

## 预定义常量

Go 语言中预定义了这些常量：

- true
- false
- iota

其中的`iota`比较特殊。

Iota （中文音译：约塔），是一个全名，而不是某一组词的缩写。 iota 是希腊字母中的第 9 个字母。

iota 是典型的数学符号，它所表示的含义如下：

1. 作为求和算法的迭代器
2. 作为[下标索引]
3. 对于[复数的虚部]

```go
const a = iota
const b = iota
const c = iota
```

这样，`a`、`b`、`c`的值分别是：0、1、2。
