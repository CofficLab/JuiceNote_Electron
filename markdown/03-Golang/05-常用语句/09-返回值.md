## 多个返回值

Go 的函数支持返回多个值：

<div class="run"></div>

```go
package main
import "fmt"

func stuff()(string,string) {
    return "老白","小六"
}

func main() {
    one,two := stuff()

    fmt.Println(one)
    fmt.Println(two)
}
```
