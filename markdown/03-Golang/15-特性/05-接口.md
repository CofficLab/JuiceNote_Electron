# 接口

## 简介

Go 接口是隐式实现。

## 类型断言

`类型断言` 可用于判断一个对象是否实现了某个接口。

<div class="run"></div>

```go
package main
import "fmt"

type Animal interface {
    shout() string
}

type Dog struct {}

func (dog *Dog) shout() string {
    return "汪汪汪"
}

func main() {
    var dog interface{}
    dog = &Dog{}
    dogAnimal,ok := dog.(Animal)

    if ok {
        fmt.Println("yes", dogAnimal)
    } else {
        fmt.Println("no")
    }
}
```
