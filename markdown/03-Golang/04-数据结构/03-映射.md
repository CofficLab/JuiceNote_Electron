# 映射

对于数组和切片，如果需要读出一个特定的值，只能从开头开始，逐个查看，直到找到目标。

映射可以让这个过程变得更简单。

一个映射是通过键来访问每一个值的集合。

## 创建（声明）

<div class="run"></div>

```go
package main
import "fmt"
import "reflect"

func main() {
    stuff := make(map[string]string)

    stuff["掌柜"] = "佟湘玉"
    stuff["厨师"] = "李大嘴"
    stuff["账房"] = "吕轻侯"
    stuff["跑堂"] = "白展堂"

    fmt.Println(reflect.TypeOf(stuff))
    fmt.Println("掌柜是",stuff["掌柜"])
}
```

<div class="run"></div>

```go
package main
import "fmt"
import "reflect"

func main() {
    ages := make(map[string]int)

    ages["佟湘玉"] = 28
    ages["李大嘴"] = 22
    ages["吕轻侯"] = 23
    ages["白展堂"] = 27

    fmt.Println(reflect.TypeOf(ages))
    fmt.Println("白展堂的年龄是", ages["白展堂"])
}
```
