# 集合

集合更接近数学上集合的概念。

集合中的元素是无序的、不重复的任意对象，可以通过集合去判断数据的从属关系，有时还可以通过集合把数据结构中重复的元素减掉。

集合不能被切片也不能被索引，除了做集合运算之外，集合元素还可以被添加和删除：

Go 没有内置集合这种数据结构。

<div class="banner">哪种编程语言内置了集合？</div>

Python 中是原生支持集合的，而且很简单：

<div class="run"></div>

```python
a = {1,2,3,4}
a.add(5)
print(a)
```

<div class="run"></div>

```python
a = {1,2,3,4}
a.discard(3)
print(a)
```
