# 包

## 导入

```go
package main
import "fmt"
import "reflect"
```

也可以这样导入多个包：

```go
package main
import (
    "fmt"
    "reflect"
)
```

## 常见报错

### 包未使用

```
imported and not used: "{包名}"
```

Go 禁止导入不使用的包，应删除导入语句。

<div class="run"></div>

```go
package main
import "fmt"
import "reflect"

func main() {
    fmt.Println("你好")
}
```
