输出

<div class="run"></div>

print('老白，吃了吗？')

格式化输出

<div class="run"></div>

name = '小六'

print("吃了吗？ %s" % name)

除了 %s，还有更多表示其他意义的占位符，在此列举部分：

类型

含义

b

将整数表示为二进制数

d

将整数表示为十进制数

e

使用科学计数法来表示小数

...

...

换行输出

<div class="run"></div>

print("老白，\n吃了吗？")

制表符输出

<div class="run"></div>

print("小六吃了吗？")
print("小六\t吃了吗？")

这里的 t 是 table 的简写，制表符主要用于对齐表格的列：

<div class="run"></div>

print("学号    \t姓名 \t语文 \t\t数学 \t\t英语")
print("2017001\t兰妃 \t99  \t\t88   \t\t0")
print("2017002\t花花 \t92  \t\t45   \t\t93")
print("2017008\t小溪 \t77  \t\t82   \t\t100")
