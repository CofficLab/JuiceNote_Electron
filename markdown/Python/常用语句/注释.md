# 注释

## 单行注释

<div class="run"></div>

```python
# 输出问候
print('你好')
```

## 多行注释

<div class="run"></div>

```python
# 输出名字
# 然后输出问候
print('老白')
print('吃了吗？')
```

另一种更常用的方式：

<div class="run"></div>

```python
'''
输出名字
然后输出问候
'''
print('老白')
print('吃了吗？')
```

使用三个双引号也可以：

<div class="run"></div>

```python
"""
输出名字
然后输出问候
"""
print('老白')
print('吃了吗？')
```
