数值

整型

整型、整数型、整数的意思相同或接近。

Python 对整数的处理非常简单和易于理解：

四则运算

<div class="run"></div>

print(2+2)
print(2-2)
print(2*3)
print(3/2)

为了进一步体会 Python 的简洁，你可以看看 Java 对这些是怎么处理的。

乘方运算

<div class="run"></div>

print(3**2)
print(2**3)
print(2*3)
print(10**5)

运算次序

可以使用括号来表示运算次序，和平时数学中的用法一致：

<div class="run"></div>

print((2 + 3) * 4)

浮点型

浮点数、浮点型意思相同。

在 Python 中，带小数点的数字都称为浮点数。

Python 对浮点数的处理也是符合人类的思维直觉的。

四则运算

<div class="run"></div>

print(0.1 + 0.1)
print(0.2 - 0.1)
print(2 * 0.1)

对于浮点数，所有编程语言都会存在精度问题：

<div class="run"></div>

print(0.2 + 0.1)

输出的数值不是 0.3，但是非常接近 0.3，这与计算机存储浮点数的方式有关。

对于精度要求分毫不差的业务，不要使用浮点数，比如：

某件商品的价格是 0.1 元，用户购买了 3 件，在设计系统时可将商品价格设置为 10 分。

复数型

Python 没有内置复数类型。
