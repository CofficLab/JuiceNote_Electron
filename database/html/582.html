字符串

在Redis中,strings是最简单的一种数据结构。它们是二进制安全的,这意味着它们可以包含任何数据 - 文本,整数,对象等。strings的一些主要属性:• 最大大小为512MB。
• 值可以递增/递减(使用INCR,DECR命令)。这在实现计数器时很有用。
• 可用于缓存键或缓存值。
• 快速的读/写速度(Redis是一个内存数据库)。一些常用的字符串操作命令:• SET key value - 将字符串键设置为保存字符串值。
• GET key - 获取键key的值。
• INCR key - 将键key的数字值增加1。
• DECR key - 将键key的数字值减少1。
• INCRBY key increment - 将键key的数字值增加增量的量。
• APPEND key value - 追加一个值到键。
• STRLEN key - 获取存储在键key中的值的长度。字符串的示例用法:

# 设置用户配置文件数据
HMSET user:1 username john age 30  
HGETALL user:1
# 返回:
# username john  
# age 30

# 增加年龄  
HINCRBY user:1 age 1
HGET user:1 age  
# 返回31

# 检查是否存在username字段
HEXISTS user:1 username
# 返回1  

# 删除age字段
HDEL user:1 age  
HGETALL user:1   
# 返回:  
# username john
