空白标识符_

blank identifier

概述

_ 是一个特殊的标识符，一般位于键盘上等号左侧的那个键，被称为空白标识符。

0 - =

示例

避免 “变量未使用” 错误

当定义一个变量后，一直未使用 ，为了避免报错，可以将其赋值给空白标识符。

package main

func main() {
    n := 1024
    _ = n // 如果没有这行代码，就会报错 Unused variable 'n'
}

忽略错误

package main
import "strconv"
import "fmt"

func main() {
    n, _ := strconv.Atoi("1")    // 函数的第 2 个返回值是一个错误类型值
    fmt.Println(n)
}

总结

它可以像其他标识符那样用于变量的声明或赋值（任何类型都可以赋值给它），但任何赋给这个标识符的值都将被丢弃， 因此这些值不能在后续的代码中使用，也不可以使用这个标识符作为变量对其它变量进行赋值或运算
