循环

Go 语言中只有一种循环：for 循环。

for 循环

package main
import "fmt"

func main() {
    for i := 1; i <= 10; i++ {
        fmt.Println(i)
    }
}

i++等价于i = i + 1，很多编程语言都是这样。

其他循环

你可能在其他编程语言中见过其他类型的循环：

  • while

  • do...while

Go 中没有这些循环。

想了解多种循环方式的话，你可以看一看 PHP 的相关章节。

循环嵌套

package main
import "fmt"

func main() {
    for i := 1; i <=10; i++ {
        for j := 1; j <=10; j++ {
            fmt.Printf("%d * %d = %d \n", i, j, i*j)
        }
    }
}

跳出循环

break

package main
import "fmt"

func main() {
    for i := 1; i <= 10; i++ {
        fmt.Println(i)
        if (i == 8) {
            break
        }
    }
}

continue

continue 表示跳过本轮，直接进行下一个：

package main
import "fmt"

func main() {
    for i:= 1; i <= 10; i++ {
        if (i == 8) {
            continue
        }
        fmt.Println(i)
    }
}

无限循环

package main
import "fmt"

func main() {
    for i:= 1; true; i++ {
        fmt.Println(i)
    }
}

我们应该避免写出无限循环
