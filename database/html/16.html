多个返回值

Go 的函数支持返回多个值：

package main
import "fmt"

func stuff()(string,string) {
    return "老白","小六"
}

func main() {
    one,two := stuff()

    fmt.Println(one)
    fmt.Println(two)
}

丢弃返回值

package main
import "fmt"

func stuff()(string,string) {
    return "老白","小六"
}

func main() {
    one,_ := stuff()

    fmt.Println(one)
}

如果对某个返回值不感兴趣，可以将其赋给空白符_。
