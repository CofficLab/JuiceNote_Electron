结构体

创建一个 Date Struct 来保存年月日的值。

<div class="run"></div>

package main
import "fmt"
import "reflect"

type Date struct {
    Year int
    Month int
    Day int
}

func main() {
    date := Date{Year: 2020, Month: 5, Day: 2}

    fmt.Println(reflect.TypeOf(date))
    fmt.Print(date)
}

在 Go 中，这种自定义的数据结构叫做：结构体。
