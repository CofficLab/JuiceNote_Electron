for

概述

Go 仅提供了关键字 for 来表示循环，并没有提供 while 和 do-while 语句，这一点和主流编程语言不同。

语法规则

for 初始表达式; 条件表达式; 迭代表达式 {
    // do something
}

注意：迭代表达式中，不支持 ++i, --i 这种形式，详情见 自增 / 自减。

例子

单个计数器

package main
import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}

多个计数器

package main
import "fmt"

func main() {
    for i, j := 1, 5; i <= 5; i, j = i+1, j-1 {
        fmt.Println("i = ", i, " j = ", j)
    }
}

模仿 while

package main
import "fmt"

func main() {
    i := 0
    for i < 5 {
        fmt.Println(i)
        i++
    }
}

模仿 do-while

package main
import "fmt"

func main() {
    i := 0
    for {
        fmt.Println(i)
        i++
        if i >= 5 {
            break
        }
    }
}

无限循环

package main
import "fmt"

func main() {
    i := 0
    for {
        fmt.Println(i)
        i++
        if i >= 5 {
            break // 删除这行代码，将会进入无限循环
        }
    }
}

// 如果删除 `break` 语句，程序进入无限循环后可以使用 `Ctrl + C` 退出。
