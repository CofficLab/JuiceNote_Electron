单行多语句

package main
import "fmt"

func main() {
    apple := 8; pear := 9; banana := 10

    fmt.Println(apple)
    fmt.Println(pear)
    fmt.Println(banana)
}

缩进

Go 中使用缩进主要是为了更好的可读性。

使用缩进来标识代码块是 Python 最鲜明的特点，Go 使用大括号 {} 来实现同样的效果。

package main
import "fmt"

func main() {
    x := 22
    if (x > 20) {
        fmt.Println("yes");
    }
}
x = 22
if x > 20:
    print('yes')
